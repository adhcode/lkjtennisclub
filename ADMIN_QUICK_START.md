# 🚀 Admin Quick Start Guide

## Create Admin User (30 seconds)

```bash
# Run this command in your terminal
npm run create-admin
```

**Default Credentials:**
- Email: `admin@lkjtennisclub.com`
- Password: `Admin@123`

**Custom Credentials:**
```bash
ADMIN_EMAIL="your@email.com" ADMIN_PASSWORD="YourPass123" ADMIN_NAME="Your Name" npm run create-admin
```

## Sign In

1. Go to: http://localhost:3000/auth/signin
2. Enter admin email and password
3. You're in! 🎉

## What You Get

### Admin Sidebar (Left Side)
- Dashboard
- Members
- Orders
- Products
- Categories
- Brands
- Product Requests
- View Site
- Sign Out

### No Navbar/Footer
Admin pages have their own clean layout with just the sidebar.

## Quick Actions

### Add a Product
1. Click "Products" in sidebar
2. Click "Add New Product"
3. Fill in details
4. Upload images
5. Save

### Manage Orders
1. Click "Orders" in sidebar
2. View all orders
3. Update status
4. Process payments

### Add a Member
1. Click "Members" in sidebar
2. Click "Add New Member"
3. Fill in details
4. Generate QR code

## ⚠️ Important

**Change the default password immediately after first login!**
